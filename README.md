# 🎨 Tailwind Quest

An interactive gamified platform for learning and visualizing Tailwind CSS in real-time.

Tailwind Quest transforms traditional CSS learning into an engaging experience through live previews, interactive lessons, daily challenges, XP progression, achievements, and a free-form playground where users can experiment with Tailwind utility classes instantly.

---

## 🚀 Features

### 📚 Interactive Learning

* Structured Tailwind CSS lessons
* Instant visual feedback
* Learn utility classes through experimentation
* Beginner-friendly interface

### 🎮 Gamification System

* XP and leveling system
* Daily streak tracking
* Achievement badges
* Progress saving using Local Storage
* Lesson completion rewards

### 🏆 Daily Challenges

* New Tailwind challenges every day
* Test your knowledge with practical tasks
* Earn bonus XP for correct solutions

### 🛠️ Free Type Playground

* Type any Tailwind CSS utility class
* Real-time visual preview
* Perfect for experimentation and rapid prototyping

### 📱 Responsive Design

* Fully optimized for desktop, tablet, and mobile devices
* Clean and modern UI
* Built with accessibility and usability in mind

### 💾 Persistent Progress

* XP, levels, streaks, badges, and completed lessons are automatically saved
* Continue learning where you left off

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/subozlame/code4fun.git
```

Navigate into the project:

```bash
cd code4fun
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
